\subsection{网络连接的方式}
\begin{enumerate}
\item 直接连接的网络（直连网)
\begin{itemize}
\item \underline{点对点（point-to-point）网络：}包括专用介质、节点/主机
\begin{itemize}
	\item 单向（simplex）：如广播、电视
	\item 半双工（half duplex）：异步双向，如对讲机
	\item 全双工（full duplex）: 同步双向，如电话
\end{itemize}
\end{itemize}
\end{enumerate}
